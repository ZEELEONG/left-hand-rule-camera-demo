左手定律摄像头演示 v3.3

这是一个纯静态网页项目，可以本地运行，也可以部署到 GitHub Pages 后通过网页链接访问。

使用方法：
1. 将整个文件夹解压到带摄像头的 Windows 电脑上。
2. 双击 start-windows.bat。
3. 浏览器打开后，点击“打开摄像头”。
4. 浏览器询问摄像头权限时，选择“允许”。
5. 面向摄像头展开左手：四指表示电流 I，穿入掌心表示磁场 B，拇指表示安培力 F。

注意：
- 推荐使用 Chrome 或 Edge。
- 请通过 start-windows.bat 打开，不要直接双击 index.html；摄像头权限在 localhost 下更稳定。
- v3.3 已内置 MediaPipe Hands classic、wasm、Three.js 和手部识别模型，启动识别不再依赖 jsDelivr/unpkg。
- v3.3 使用 Three.js 3D 坐标系渲染电流 I、磁场 B、安培力 F。
- v3.3 使用食指根部、小指根部、手掌根部三点建立稳定掌面。
- v3.3 将四指平均方向投影到掌面作为蓝色电流方向。
- v3.3 修正镜像模式下画面、骨架和 Three.js 层的坐标对齐方式。
- v3.3 改用屏幕几何判断左右手：当前画面里像左手才进入演示，像右手则提示抬起左手。
- 蓝色 3D 标签外移到 0.2 倍箭头长度，黄色 3D 标签外移到 0.4 倍箭头长度。
- 蓝色/黄色箭头长度保持 1.5 倍，红色磁场阵列间距保持放大一倍，红色箭头长度保持翻倍。
- 磁场 B 的 3x3 阵列会随手掌三维翻转自然缩短和旋转。
- 如果页面左上角不是 v3.3，说明打开的仍是旧版本文件，请重新解压最新 zip。

网页托管：
- 推荐部署到 GitHub Pages，手机和 iPad 通过 HTTPS 链接访问即可使用摄像头。
- 详细步骤见 DEPLOY-GITHUB-PAGES.md。
